ขั้นตอนการสร้าง Measures สำหรับคำนวณร้อยละความครอบคลุมวัคซีน
เราจะแบ่งการสร้าง Measure ออกเป็น 3 ส่วน เพื่อให้ง่ายต่อการตรวจสอบความถูกต้องของข้อมูล (Cross-check) ครับ

1. สร้าง Measure สำหรับตัวหาร (B: เป้าหมาย)
นับผลรวมจากคอลัมน์ จำนวนประชากร ในตารางเป้าหมาย

ข้อมูลโค้ด
B_Target = 
SUM('ชื่อตารางเป้าหมาย'[จำนวนประชากร])
2. สร้าง Measure สำหรับตัวตั้ง (A: ผลงาน)
นับผลรวมจากคอลัมน์ Value ในตารางประวัติการรับวัคซีน โดยกรองเฉพาะวัคซีนที่กำหนดในเกณฑ์เด็กอายุ 1 ปี

หมายเหตุ: หากคอลัมน์ Vaccine ของคุณมีเฉพาะข้อมูลวัคซีนเด็ก 1 ปีอยู่แล้ว (กรองมาตั้งแต่ Power Query) คุณสามารถตัดฟังก์ชัน CALCULATE ออก เหลือแค่ SUM('ชื่อตารางวัคซีน'[Value]) ได้เลยครับ

ข้อมูลโค้ด
A_Vaccinated = 
CALCULATE(
    SUM('ชื่อตารางวัคซีน'[Value]),
    'ชื่อตารางวัคซีน'[Vaccine] IN {
        "bcg", "hbv1", "dtp1", "dtp3", "hb3", 
        "hib3", "polio3", "mmr1", "ipv", "rota"
    } 
)
3. สร้าง Measure หาร้อยละความครอบคลุม (% Coverage)
นำผลลัพธ์จากข้อ 1 และข้อ 2 มาหารกันด้วยฟังก์ชัน DIVIDE เพื่อป้องกัน Error ในกรณีที่พื้นที่นั้นไม่มีเด็กเป้าหมายเลย (ตัวหารเป็น 0)

ข้อมูลโค้ด
% Coverage = 
DIVIDE([A_Vaccinated], [B_Target], 0)
💡 ทริคเพิ่มเติมสำหรับการตั้งค่าใน Power BI:
หลังจากกดสร้าง Measure % Coverage เสร็จแล้ว ให้คลิกที่ชื่อ Measure นี้ในหน้าต่าง Data (ด้านขวา) จากนั้นไปที่แท็บ Measure tools (เครื่องมือการวัด) ด้านบน แล้วคลิกที่สัญลักษณ์ % เพื่อแปลงรูปแบบการแสดงผลให้เป็นเปอร์เซ็นต์โดยอัตโนมัติ (เช่น 85.50%) แทนการคูณ 100 ในโค้ดครับ จะทำให้ Power BI นำค่าไปสร้างกราฟได้ถูกต้องที่สุด
