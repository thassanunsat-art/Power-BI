# 📊 Power BI DAX: คำนวณร้อยละความครอบคลุมของการฉีดวัคซีนเด็ก 1 ปี

เอกสารชุดนี้รวบรวมคำสั่ง DAX (Data Analysis Expressions) สำหรับใช้ในระบบ Power BI เพื่อประเมินผลและวิเคราะห์ข้อมูลความครอบคลุมของการได้รับวัคซีน โดยแบ่งการคำนวณออกเป็น 3 Measures เพื่อให้ง่ายต่อการตรวจสอบความถูกต้องของข้อมูล (Data Validation) และลดความซับซ้อนของโค้ด

---

## 📌 1. ตัวหาร (B: จำนวนเป้าหมาย)
คำนวณจำนวนประชากรเด็กอายุ 1 ปีทั้งหมดในพื้นที่รับผิดชอบ (อ้างอิงจากตารางเป้าหมาย)

```dax
B_Target = 
SUM('ชื่อตารางเป้าหมาย'[จำนวนประชากร])
```

📌 2. ตัวตั้ง (A: จำนวนผลงาน)
คำนวณจำนวนเด็กเป้าหมายที่ได้รับวัคซีนครบตามเกณฑ์ (BCG, HBV1, DTP1, DTP3, HB3, Hib3, Polio3, MMR1, IPV, Rota)

Note: หากมีการจัดการฐานข้อมูล หรือกรองข้อมูลวัคซีนเฉพาะเด็ก 1 ปีมาแล้วจาก Power Query สามารถตัดฟังก์ชัน CALCULATE และชุดเงื่อนไข IN ออก เพื่อให้การประมวลผลเร็วขึ้นได้เลย

```
A_Vaccinated = 
CALCULATE(
    SUM('ชื่อตารางวัคซีน'[Value]),
    'ชื่อตารางวัคซีน'[Vaccine] IN {
        "bcg", "hbv1", "dtp1", "dtp3", "hb3", 
        "hib3", "polio3", "mmr1", "ipv", "rota"
    } 
)
```
📌 3. ร้อยละความครอบคลุม (% Coverage)
คำนวณสัดส่วนผลงานต่อเป้าหมาย โดยใช้ฟังก์ชัน DIVIDE เพื่อจัดการกับกรณีที่ตัวหาร (พื้นที่ที่ไม่มีเด็กเป้าหมาย) เป็น 0 ซึ่งจะช่วยป้องกัน Error แบบ Infinity หรือ NaN บน Dashboard

```
% Coverage = 
DIVIDE([A_Vaccinated], [B_Target], 0)
```
💡 การตั้งค่าการแสดงผล (Formatting)
หลังจากสร้าง Measure % Coverage เรียบร้อยแล้ว แนะนำให้คลิกที่ชื่อ Measure ไปที่แท็บ Measure tools ใน Power BI และตั้งค่า Format ให้เป็น Percentage (%) เพื่อให้กราฟและรายงานแสดงผลตัวเลขได้อย่างถูกต้องตามมาตรฐาน
